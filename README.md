# Mark Romanov

### Senior Frontend Engineer

Senior Frontend Developer with 5+ years of commercial experience in fintech and product development.

I build scalable SPA and SSR applications using React and TypeScript.  
Strong background in high-load systems, offline-first architecture, backend-driven UI (BDUI), and shared UI libraries used across multiple teams.

Comfortable working both on feature delivery and on frontend architecture:
modular structuring, state management strategy, performance optimization,
data-layer design, and cross-team standardization of components and practices.

I focus on predictable architecture, controlled technical debt,
and careful handling of asynchronous flows and application state.

📍 GMT+3
